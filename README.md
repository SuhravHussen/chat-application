
# Chat-Application

### This is real time chat application. In this application a user can chat with another user by creating conversation. 

***Only Admin can add new users***

# Libraries Used
- Expree Js
- bcrypt
- express-validator
- cookie-parser
- dotenv
- ejs
- eslint
- prettier
- multer
- nodemone
- mongoose
- http-errors
- jsonwebtoken
- moment 
- socket.io

# Database used
- mongoDB
